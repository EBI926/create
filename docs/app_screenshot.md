# アプリケーションのスクリーンショット

このファイルでは、アプリケーションの実行時の様子を説明します。

## メイン画面

アプリケーションを起動すると、以下のようなインターフェースが表示されます：

```
+-----------------------------------------------+
|            請求書計算アプリ                   |
+-----------------------------------------------+
|                                               |
|        [  PDFファイルを選択  ]                |
|        選択されたファイル: 0                  |
|                                               |
+-----------------------------------------------+
|              計算結果                         |
|                                               |
| 処理済みファイル: 0       合計金額: ¥0        |
|                                               |
| +-------------------------------------------+ |
| |                                           | |
| |                                           | |
| |                                           | |
| |                                           | |
| +-------------------------------------------+ |
|                                               |
+-----------------------------------------------+
```

## ファイル選択後

PDFファイルを選択すると、以下のように結果が表示されます：

```
+-----------------------------------------------+
|            請求書計算アプリ                   |
+-----------------------------------------------+
|                                               |
|        [  PDFファイルを選択  ]                |
|        選択されたファイル: 2                  |
|                                               |
+-----------------------------------------------+
|              計算結果                         |
|                                               |
| 処理済みファイル: 2       合計金額: ¥35,000   |
|                                               |
| +-------------------------------------------+ |
| | 請求書1.pdf                               | |
| | 合計: ¥20,000                             | |
| | 検出された金額: ¥20,000                   | |
| +-------------------------------------------+ |
| | 請求書2.pdf                               | |
| | 合計: ¥15,000                             | |
| | 検出された金額: ¥10,000, ¥5,000           | |
| +-------------------------------------------+ |
|                                               |
+-----------------------------------------------+
```

## 使用方法

1. `npm install` で依存関係をインストール
2. `npm start` でアプリケーションを起動
3. 「PDFファイルを選択」ボタンをクリックして請求書をアップロード
4. アプリケーションが自動的に金額を抽出し、合計を表示
