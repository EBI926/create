# 請求書計算アプリ (Invoice Calculator)

PDFで届く複数の請求書をアップロードして、合計額を自動で算出するデスクトップアプリケーションです。

## 機能

- 複数のPDF請求書をアップロード
- 請求書から金額を自動抽出
  - 様々な金額表記形式に対応（¥123,456、123,456円、JPY 123,456など）
  - 「合計」「Total」「請求金額」などのラベルに近い金額を優先的に抽出
  - マイナス金額（値引き）の正しい処理
- 合計金額の自動計算
- シンプルで使いやすいインターフェース

## インストール方法

### 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/EBI926/create.git
cd create

# 依存関係をインストール
npm install

# アプリケーションを起動
npm start
```

### ビルド方法

```bash
# Windows/Mac用のパッケージを作成
npm run build
```

ビルドされたアプリケーションは `dist` ディレクトリに生成されます。

## 使用方法

1. アプリケーションを起動します
2. 「PDFファイルを選択」ボタンをクリックします
3. 計算したい請求書のPDFファイルを選択します（複数選択可能）
4. アプリケーションが自動的に金額を抽出し、合計を表示します

## テスト方法

### ユニットテスト

```bash
# テストを実行
npm test

# テストを監視モードで実行
npm run test:watch
```

### テスト用PDFの生成

テスト用のPDFファイルを生成するには、以下のコマンドを実行します：

```bash
# PDFKit依存関係をインストール
npm install pdfkit --save-dev

# テスト用PDFを生成
node scripts/generate-test-pdfs.js
```

生成されたPDFファイルは `test/fixtures` ディレクトリに保存されます。

## PDF金額抽出の仕組み

このアプリケーションは、以下の手順でPDFから金額を抽出します：

1. PDF.jsを使用してPDFからテキストを抽出
2. 複数の正規表現パターンを使用して様々な金額表記を検出
   - `¥123,456`、`￥123,456`、`JPY 123,456`、`123,456円` など
3. 「合計」「Total」「請求金額」などのラベルに近い金額を優先的に抽出
4. マイナス金額（値引き）を考慮して合計金額を計算

## 注意事項

- 現在のバージョンでは、日本語の請求書で「¥」「￥」「JPY」の後に数字が続く形式、または数字の後に「円」が続く形式の金額を認識します
- より複雑な請求書フォーマットの場合、認識精度が低下する可能性があります
