# 請求書計算アプリ (Invoice Calculator)

PDFで届く複数の請求書をアップロードして、合計額を自動で算出するデスクトップアプリケーションです。

## 機能

- 複数のPDF請求書をアップロード
- 請求書から金額を自動抽出
- 合計金額の自動計算
- シンプルで使いやすいインターフェース

## インストール方法

### 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/EBI926/create.git
cd create

# 依存関係をインストール
npm install

# アプリケーションを起動
npm start
```

### ビルド方法

```bash
# Windows/Mac用のパッケージを作成
npm run package
```

ビルドされたアプリケーションは `dist` ディレクトリに生成されます。

## 使用方法

1. アプリケーションを起動します
2. 「PDFファイルを選択」ボタンをクリックします
3. 計算したい請求書のPDFファイルを選択します（複数選択可能）
4. アプリケーションが自動的に金額を抽出し、合計を表示します

## 注意事項

- 現在のバージョンでは、日本語の請求書で「¥」または「JPY」の後に数字が続く形式の金額を認識します
- より複雑な請求書フォーマットの場合、認識精度が低下する可能性があります
